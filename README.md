# simple-frontend-starter-kit
A simple starter kit for a frontend app

Requirements
---

- Node.js `^6.3.0`
- Yarn `^0.19.1`
- Phantomjs `^2.1.1`

Initialize
---

```sh
$ yarn
```

Develop your application
---

```sh
$ yarn dev
```

Check types
---

```sh
$ yarn type
```

Test your application
---

```sh
$ yarn test
```

Continuously test your application while you develop it
---

```sh
$ yarn test:dev
```

Build your application
---

```sh
$ yarn build
```
